# Alce's Barbearia - Piloto Flutter

Este projeto de piloto vai usar a identidade da Alce's Barbearia como base oficial.

## Marca

- Nome: `Alce's Barbearia`
- Papel no projeto: cliente piloto
- Direcao visual: premium, escura, elegante e direta

## Assets Ja Existentes

Os arquivos de logo ja estao no projeto:

- `public/assets/Logo_Alces_Barbershop.png`
- `public/assets/Logo_Alces_Barbershop-1.png`
- `public/assets/Logo_Alces_Barbershop-1-1.png`

## Identidade Visual

- fundo escuro
- dourado como cor principal
- branco suave para texto
- verde para sucesso
- vermelho para erro

## Referencias Do Piloto

- unidade principal: `Alce's Barbearia - Matriz`
- outras unidades de referencia: `Escola Agrícola` e `Gaspar`
- tom de interface: profissional, premium e simples

## Uso No Flutter

O app Flutter deve usar:

- logo da Alce's no splash e no header
- nome da marca como identidade principal
- unidades e servicos com nomes da Alce's
- linguagem de checkout, assinatura e agendamento baseada no material de referencia

## Regra

Nao usar marca genérica no piloto. Tudo que for visual ou textual deve reforcar a Alce's Barbearia como cliente real.
